Node.js, execução JavaScript!

Linguagem que permite soluções em tempo real, trabalhando no mundo moderno de forma poderosa e barata para a criação e a manutenção de ambientes de tecnologia com altas demandas, criada pelo engenheiro de software Ryan Dahl.

Seu propósito

O Node.js é um ambiente de execução do código JavaScript do lado servidor (server side), que na prática se reflete na possibilidade de criar aplicações standalone (autossuficientes) em uma máquina servidora, sem a necessidade do navegador. Este nasceu em 2009 como uma resposta às tentativas de rodar códigos JavaScript em modo server side, uma vez que a linguagem tinha como meta principal a manipulação do DOM (Document Object Model) e deixar as aplicações web mais interativas e dinâmicas.
